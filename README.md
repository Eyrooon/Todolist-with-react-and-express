# Todolist-with-react-and-express


#clone
- git clone https://github.com/Eyrooon/Todolist-with-react-and-express.git
- cd todolist-with-react-and-express

#Run Backend
- npm install
- nodemon index


#Run Frontend
- cd client/todos
- npm install
- npm start

#Database
- Make sure you have Mongodb on your machine
- open terminal
- type mongod
